# Empty but Live — FlyRank AI Internship

A minimal React + TypeScript project created for the Week 4 assignment.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Import this repository into Vercel or Netlify. Use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`

After deployment, open the public URL on a second device and take a screenshot.
